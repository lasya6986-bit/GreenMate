# 🌱 GreenMate - Smart Plant Care Companion

GreenMate is a beginner-friendly Python project developed to help users manage and care for their plants efficiently.

This project uses Python and SQLite to store plant details such as:
- Plant Name
- Last Watered Date
- Sunlight Requirement

## 🚀 Features

- Add Plant
- View Plants
- Update Watering Date
- Delete Plant
- Search Plant
- SQLite Database Integration

## 🛠️ Technologies Used

- Python
- SQLite3

## 📌 Future Improvements

- Watering reminders
- GUI/Web Application
- Plant disease detection using Machine Learning
- Smart plant recommendations
- User login system
- Image-based plant recognition

## ▶️ How to Run

1. Run `database.py`
2. Run `main.py`

## 💡 Project Goal

GreenMate is being developed as a scalable long-term project that can later integrate Machine Learning features for smarter plant care assistance.
